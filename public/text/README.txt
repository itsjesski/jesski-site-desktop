Welcome to Jess's Desktop!

This is a Windows-style desktop interface built with React and TypeScript.

Getting Started:
- Double-click on icons to open applications
- Drag windows around to organize your workspace
- Use the taskbar to switch between open applications
- Share any window or your entire desktop state using the share button

Settings & Customization:
To access settings, look for the GEAR (⚙️) icon in the system tray (bottom-right corner of the taskbar, next to the clock).

Features:
- Draggable and resizable windows
- Desktop icons
- Taskbar with open applications
- Various applications (About, Text viewer, Projects, Stickers)
- Sound effects and audio controls
- Shareable URLs for desktop states

Enjoy exploring!
